## Emergency procedures

<Card 
  title="Emergency unlock via backplate" 
  icon="triangle-exclamation" 
  href="/lkbx/support/emergency-backplate"
>
  As a last resort, learn how to physically access your key by breaking the backplate. This method is destructive and requires a replacement part.
</Card>

<Warning>
The backplate emergency unlock is a destructive procedure. Always try the software emergency unlock first. Only use the physical method when the software unlock is unavailable or has been disabled.
</Warning>
