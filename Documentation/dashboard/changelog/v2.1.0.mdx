<Update label="Version 2.1.0" description="Released February 20, 2026">

## Keyholder Experience Improvements

- **Unified lock interface for keyholders** — Keyholders now use the same keyholder selection dropdown as lockees when initiating lock sessions, with context-appropriate options (self-lock and partner management links hidden for keyholders)
- **Keyholder auto-selection** — When a keyholder initiates a lock session, they are automatically pre-selected in the keyholder list for a smoother workflow
- **Keyholder visibility fix** — Keyholders can now see all of a lockee's connected partners in settings, not just themselves, enabling proper multi-keyholder coordination

</Update>
