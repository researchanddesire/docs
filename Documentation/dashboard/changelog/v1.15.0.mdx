<Update label="Version 1.15.0" description="Released February 13, 2026">

## Community Templates Now Free

- **Free template imports** — All users can now import and use community templates from the Explore page without requiring an Ultra subscription. The "Ultra Required" restriction has been removed, making template sharing accessible to everyone.

## Improvements

- **Navigation layout cleanup** — Refactored sidebar navigation structure for improved code maintainability
- **Avatar loading state** — Fixed text sizing on avatar skeleton loading states for better visual consistency
- **ESLint configuration** — Added .cursor directory to lint ignore list for cleaner development tooling

## Database

- **Safer legacy user migration** — Improved migration script to handle duplicate emails gracefully and auto-execute migration for legacy users
