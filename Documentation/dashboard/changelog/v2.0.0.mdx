- **DTT active toy persistence** — Fixed training sessions incorrectly losing the selected toy; the toy set at session start now persists through completion
- **Sidebar layout prototype** — Added new collapsible sidebar with device switcher for navigating between DTT, Lockbox, and OSSM devices
- **Seed data improvements** — Enhanced local development setup with additional test users, trainers, and OSSM devices for better testing coverage

</Update>

<Update label="Version 2.0.0" description="Released February 19, 2026">

## Duration Calculation Overhaul

- **Countdown timer accuracy** — Rewrote duration calculation to use hybrid calendar-aware and millisecond arithmetic, eliminating negative sub-components caused by month boundary clamping and DST transitions
- **Month boundary fix** — Fixed edge case where timers would display incorrect values when crossing months with different lengths (e.g., January 30 to February 28)
- **Long duration support** — Improved handling of multi-year intervals with proper preservation of years and months units
- **Zero duration handling** — Duration calculations now correctly return empty results when start and end times are identical

</Update>
