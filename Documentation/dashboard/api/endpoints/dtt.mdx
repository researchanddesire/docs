## List DTT Devices

<div className="not-prose">
  `GET /api/v1/dtt`
</div>

Returns all Deepthroat Trainer devices you have access to.

<ResponseField name="data" type="object[]" required>
  Array of DTT device objects.

  <ResponseField name="id" type="number">
    Device ID.
  </ResponseField>

  <ResponseField name="bubbleId" type="string">
    Unique device identifier.
  </ResponseField>

  <ResponseField name="ownerId" type="number">
    ID of the device owner.
  </ResponseField>

  <ResponseField name="macAddress" type="string">
    Device MAC address.
  </ResponseField>

  <ResponseField name="serialNumber" type="number">
    Hardware serial number.
  </ResponseField>

  <ResponseField name="softwareVersion" type="string">
    Currently installed firmware version.
  </ResponseField>

  <ResponseField name="lastVisited" type="string">
    ISO 8601 timestamp of the last device connection.
  </ResponseField>

  <ResponseField name="apiHits" type="number">
    Total API requests made by this device.
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dashboard.researchanddesire.com/api/v1/dtt
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "ok": true,
  "data": [
    {
      "id": 1,
      "bubbleId": "dtt-abc",
      "ownerId": 42,
      "macAddress": "AA:BB:CC:DD:EE:FF",
      "serialNumber": 1001,
      "softwareVersion": "2.1.0",
      "lastVisited": "2026-02-17T12:00:00Z",
      "apiHits": 256
    }
  ]
}
```
</ResponseExample>

## List DTT Settings

<div className="not-prose">
  `GET /api/v1/dtt/settings`
</div>

Returns all training program settings (templates) you have access to, **without** segments. Use this for an overview of available programs.

<ResponseField name="data" type="object[]" required>
  Array of settings objects.

  <ResponseField name="id" type="number">
    Settings ID.
  </ResponseField>

  <ResponseField name="name" type="string">
    Program name.
  </ResponseField>

  <ResponseField name="description" type="string">
    Program description.
  </ResponseField>

  <ResponseField name="isPublic" type="boolean">
    Whether the program is publicly shared.
  </ResponseField>

  <ResponseField name="isActive" type="boolean">
    Whether this is the currently active program.
  </ResponseField>

  <ResponseField name="handsFreeMode" type="boolean">
    Whether hands-free mode is enabled.
  </ResponseField>

  <ResponseField name="targetDepth" type="number">
    Target depth setting in millimeters.
  </ResponseField>

  <ResponseField name="targetWindow" type="number">
    Tolerance window for depth detection.
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dashboard.researchanddesire.com/api/v1/dtt/settings
```
</RequestExample>

## Get DTT Settings with Segments

<div className="not-prose">
  `GET /api/v1/dtt/settings/{id}`
</div>

Returns a specific training program **including all segments**. Segments define the individual phases of a training session.

<ParamField path="id" type="number" required>
  The settings (program) ID.
</ParamField>

<ResponseField name="data" type="object" required>
  The settings object with a nested `Segment` array.

  <ResponseField name="Segment" type="object[]">
    Array of training segments in order.

    <ResponseField name="id" type="number">
      Segment ID.
    </ResponseField>

    <ResponseField name="type" type="string">
      Segment type.
    </ResponseField>

    <ResponseField name="duration" type="number">
      Duration in seconds.
    </ResponseField>

    <ResponseField name="repeat" type="number">
      Number of times to repeat this segment.
    </ResponseField>
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dashboard.researchanddesire.com/api/v1/dtt/settings/5
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "ok": true,
  "data": {
    "id": 5,
    "name": "Beginner Program",
    "isActive": true,
    "handsFreeMode": false,
    "targetDepth": 50,
    "Segment": [
      {
        "id": 10,
        "type": "warmup",
        "duration": 30,
        "repeat": 1
      },
      {
        "id": 11,
        "type": "training",
        "duration": 60,
        "repeat": 3
      }
    ]
  }
}
```
</ResponseExample>
