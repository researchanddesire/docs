## List Users

<div className="not-prose">`GET /api/v1/users`</div>

Returns all users you have access to. Results are filtered by your account permissions — you'll see yourself and any users connected to you via partnerships.

<ResponseField name="ok" type="boolean" required>
  Always `true` on success.
</ResponseField>

<ResponseField name="data" type="object[]" required>
  Array of user objects.

<ResponseField name="id" type="number">
  The user's internal ID.
</ResponseField>

<ResponseField name="bubbleId" type="string">
  Unique public identifier for the user.
</ResponseField>

<ResponseField name="name" type="string">
  Display name.
</ResponseField>

<ResponseField name="accountType" type="string">
  One of `user`, `support`, `admin`, or `superadmin`.
</ResponseField>

<ResponseField name="league" type="number">
  The user's current league tier.
</ResponseField>

  <ResponseField name="rank" type="number">
    The user's leaderboard rank.
  </ResponseField>
</ResponseField>

<RequestExample>
  ```bash cURL curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dashboard.researchanddesire.com/api/v1/users ```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "ok": true,
  "data": [
    {
      "id": 42,
      "bubbleId": "abc-123",
      "name": "Alice",
      "accountType": "user",
      "league": 3,
      "rank": 150
    }
  ]
}
```
</ResponseExample>

## Get User by ID

<div className="not-prose">`GET /api/v1/users/{id}`</div>

Returns detailed information about a specific user, including their profile data. You can only access users you have a connection with.

<ParamField path="id" type="number" required>
  The user's internal ID.
</ParamField>

<ResponseField name="data" type="object" required>
  User object with extended fields.

<ResponseField name="id" type="number">
  The user's internal ID.
</ResponseField>

<ResponseField name="bubbleId" type="string">
  Unique public identifier.
</ResponseField>

<ResponseField name="name" type="string">
  Display name.
</ResponseField>

<ResponseField name="accountType" type="string">
  Account type.
</ResponseField>

<ResponseField name="totalPoints" type="number">
  Lifetime points earned.
</ResponseField>

<ResponseField name="matchPoints" type="number">
  Points earned in the current match period.
</ResponseField>

  <ResponseField name="Profile" type="object">
    The user's public profile, if accessible.
  </ResponseField>
</ResponseField>

<RequestExample>
  ```bash cURL curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://dashboard.researchanddesire.com/api/v1/users/42 ```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "ok": true,
  "data": {
    "id": 42,
    "bubbleId": "abc-123",
    "name": "Alice",
    "accountType": "user",
    "league": 3,
    "rank": 150,
    "totalPoints": 12500,
    "matchPoints": 800,
    "Profile": {
      "bio": "Hello world"
    }
  }
}
```

```json 404 Not Found
{
  "ok": false,
  "error": "User not found"
}
```

</ResponseExample>
