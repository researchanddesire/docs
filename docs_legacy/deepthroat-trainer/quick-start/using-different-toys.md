# Assign Your Default Toy

### Set your default toy in the Dashboard

When starting out with the Deepthroat Trainer, you need to set your default toy in the Dashboard so the device can give you accurate feedback based on the toy's size.

{% stepper %}
{% step %}

#### Log in to your Dashboard and open your Toybox

See the earlier steps on how to get to your Dashboard. Once you're signed in, under the "Deepthroat Trainer" section on the left, click "Toybox"
{% endstep %}

{% step %}

#### Set the default R+D toy (or add your own)

If you bought one of the Research and Desire toys with your Deepthroat Trainer, you can click "Set default" next to the one that corresponds to yours. Otherwise, if you have your own toy, you can add it to your Toybox by clicking "Add new".
{% endstep %}
{% endstepper %}

This default toy will be automatically selected when creating new training templates, and you can assign different toys to specific templates if needed.

It's that easy!

Remember to update your default toy in the Dashboard if you ever want to swap out the one you are using on the device!
