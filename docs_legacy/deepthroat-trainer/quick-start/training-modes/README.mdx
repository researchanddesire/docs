## Segment Types

Training sessions are built from segments. Each segment type offers a different training approach to help develop skills progressively.

<CardGroup cols={3}>
  <Card title="Repetition" icon="repeat" href="repetition">
    Practice consistent depth and rhythm with repeated strokes at a set pace.
  </Card>

  <Card title="Endurance" icon="stopwatch" href="endurance">
    Build stamina by holding at depth for extended periods.
  </Card>

  <Card title="Freeform" icon="wand-magic-sparkles" href="freeform">
    Train with dynamic, unpredictable patterns for advanced practice.
  </Card>
</CardGroup>

<Tip>
  Start with Repetition segments to build foundational skills, then progress to Endurance and Freeform as comfort and confidence grow.
</Tip>
