# Public Links Opt-Out

## Overview

When you enable **Public Link**, your lock appears in the public voting directory by default. You can opt out to make it accessible only via direct link.

## How It Works

**Default (Public Directory):**

- ✅ Lock appears in public directory + direct link works
- Anyone can discover and vote on your lock

**Opt-Out (Private Link Only):**

- 🔒 Lock only accessible via direct link
- Share with specific people only

## How to Configure

1. Go to **Chastity Lockbox** → **Settings**
2. Enable **Public Link** toggle
3. Click **Settings icon** (⚙️) next to the toggle
4. Toggle **"Show in Public Directory"** off to opt out

## Key Points

- Direct links always work regardless of setting
- Only affects discoverability, not lock functionality
- Setting is applied when you start the lock session
