## Prerequisites

Before you begin, make sure you have:

- Your Chastity Lockbox powered on
- The lockbox connected to Wi-Fi (check the indicator in the top right of the display)
- Access to the dashboard on a computer or mobile device

## Find your pairing code

<Steps>
<Step title="Open the pairing menu">
  Turn on your Chastity Lockbox and scroll through the main menu to **Settings**, then select **Pairing**.
  
  <Note>
  The pairing code regenerates each time the device restarts, so you'll see a new code after every power cycle.
  </Note>
</Step>

<Step title="Locate the pairing code">
  At the top of the screen, you'll see a six-digit alphanumeric pairing code (letters and numbers).
  
  You have two options:
  - **Scan the QR code** displayed on screen — this takes you directly to the dashboard and attempts to pair automatically
  - **Note the code manually** and enter it in the dashboard yourself
</Step>
</Steps>

## Pair on the dashboard

<Steps>
<Step title="Navigate to device settings">
  Go to **Settings** → **Devices** in the dashboard.
  
  <Tip>
  If you scanned the QR code from your lockbox, you'll land on this page automatically with the pairing code pre-filled.
  </Tip>
</Step>

<Step title="Enter the pairing code">
  Type your six-digit pairing code into the input field and click **Pair**.
</Step>

<Step title="Confirm the pairing">
  Once paired, the device appears in your account and you can configure its settings directly from the dashboard.
  
  <Check>
  Your Chastity Lockbox is now paired and ready to use.
  </Check>
</Step>
</Steps>
