# Flashing your OSSM/RADR

OSSM and RADR are flashed using the web flasher utility here:\
[https://dashboard.researchanddesire.com/app/tools/web-flasher](https://dashboard.researchanddesire.com/app/tools/web-flasher)\
\* _Requires browser with Web Serial API support (Chrome, Edge, or Safari)_



USB UART Drivers, if needed, can be found from silabs\
Windows direct link: [https://www.silabs.com/documents/public/software/CP210x\_Windows\_Drivers.zip](https://www.silabs.com/documents/public/software/CP210x_Windows_Drivers.zip)\
Others: [https://www.silabs.com/software-and-tools/usb-to-uart-bridge-vcp-drivers?tab=downloads](https://www.silabs.com/software-and-tools/usb-to-uart-bridge-vcp-drivers?tab=downloads)
